# Config Vim

## Prerequisites

### RHEL

- Install `fzf`, the fuzzy finder

  ```bash
  git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf
  ~/.fzf/install
  ```

- Install the Silver Searcher (Ag tool)

  ```bash
  yum install epel-release
  yum install the_silver_searcher
  ```
