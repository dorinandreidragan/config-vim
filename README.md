# vim-config

This repository contains scripts to help you get started with already configured `vim`.

## Linux

### Prerequisites

The following should be installed:

 - cmake
 - npm
 - ripgrep

  ```bash
  sudo apt install -y \
    cmake \
    npm \
    ripgrep
  ```
